# AI Research Assistant

An AI-powered research assistant built using Python and Hugging Face Transformers.

## Features

- AI Question Answering
- Internet Search Integration
- Research-Based Responses
- Simple Command Line Interface
- Fast and Lightweight

## Technologies Used

- Python
- Transformers
- Hugging Face
- DDGS Search
- NLP

## How to Run

1. Install Python
2. Install Libraries

pip install transformers
pip install torch
pip install ddgs

3. Run Project

python AI-Research-Assistant.py

## Example Questions

- Who is the Prime Minister of India?
- Top AI research papers
- Explain Machine Learning
- What is Artificial Intelligence?
